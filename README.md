### Getting Started
To get started
Clone this repository: 
```bash
  git clone https://github.com/aanfadhil5/starter-component.git
```
Install the dependencies
```bash
  npm install
  or
  yarn install
```
Start the development server
```bash
  npm run dev
  or
  yarn dev
```
